### 861. Score After Flipping Matrix
1. 题意：给出由0、1构成的二维数组，每次可翻转一行或一列，翻转指的是all0变1，all1变0，求最后按行相加最大值
2. 思路：

__将一行或者一列翻转两次及以上是没有意义的，因为翻转两次等价于没有进行任何翻转。因此每一行和每一列最多被翻转了一次。__

先按行翻，以0开头的要翻

再按列翻，0个数多的要翻
```
//要数一列的1的个数，1的来源有两个
① 0....0  翻转成   1....1
② 1....1  不变
因为0^0 = 0, 1^1 = 0,
所以(一列的1的个数) == ((这一列^第一列)中0的个数)
//要数一列的0的个数，0的来源有两个
① 0....1  翻转成   1....0
② 1....0  不变
因为0^1 = 1, 1^0 = 1,
所以(一列的0的个数) == ((这一列^第一列)中1的个数)
col += A[r][c] ^ A[r][0];
```
